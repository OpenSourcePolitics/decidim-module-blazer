# Decidim::Blazer

Business intelligence made simple.

## Usage

Blazer will be available as a Component for a Participatory
Space.

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'decidim-blazer
```

And then execute:

```bash
bundle
```
